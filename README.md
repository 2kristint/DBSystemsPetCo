# DBSystemsPetCo

## Getting Started

### Installation

Install all the required packages in the `client` and `server` directory. Run these commands in the terminal:

```
cd client && npm i

cd server && npm i
```

You will then need to add an .env file with the variables corresponding to your own MySQL connection.

## Usage

To run the project locally

#### Client

1.  Run the client by right clicking index.html and select "Open with Live Server" if using VSCode. Alternatively, find the file in your folders and double-click.

#### Server

1.  Run the server by running `node server.js` in the `/server` directory
